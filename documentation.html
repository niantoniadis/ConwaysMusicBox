Documentation

Proposal

  • Musical melodies generated through cellular automata.
  • See what kind of melodies you can generate!
  • Tweak the melody by clicking notes to toggle them on/off!
  • View other people's creations and share your own!



Issues

The part of this project that took the longest was probably figuring out how to use the web audio API to shift the pitch of an audio clip, and how to ensure that all pitches are within the same musical key. I happened to find a really useful formula that converts frequencies to musical notes, so that saved me a lot of time on this part. Another challenge was setting up the firebase project, since I failed to add it to my first project, it took a little more time to remember how it was done in the homeworks.
